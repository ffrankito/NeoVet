## System Overview

### Description

A stateless web chatbot that answers FAQ questions for NeoVet clinic clients. The user sends a message via a web widget; the system streams a response generated by Claude claude-sonnet-4-6 using a system prompt seeded with real clinic data. No data is persisted. The only external call (besides Anthropic) is to `argentinadatos.com` for holiday detection.

## Authentication & Authorization

v1 is a public-facing widget. No authentication required.

### Rate Limiting

The `/api/chat` endpoint is rate-limited to **20 requests per minute per IP address**. Implemented via an in-memory `Map` in `src/lib/rate-limit.ts`. When exceeded, returns HTTP 429 with a user-friendly message in Spanish.

---

## Additional Features (not in original spec)

### Holiday Detection

On each request, the API calls `argentinadatos.com/v1/feriados/{year}` to check if today is a national holiday. If so, the system prompt is augmented with the holiday name and modified hours ("Feriados: 10:00–13:00 hs").

### Quick Replies

On first load, the chat widget displays 4 hardcoded FAQ suggestions to guide users (e.g., "Horarios de atención", "Servicios disponibles"). These are defined in `src/components/chat-widget.tsx`.

### Markdown Rendering

Bot responses are rendered using `react-markdown` with styled components for headings, lists, links, and code blocks.

## AI / LLM Usage

### Chat response

| Field | Value |
|---|---|
| **Model** | `claude-sonnet-4-6` |
| **Calling pattern** | `streamText()` — streaming |
| **Prompt location** | `src/app/api/chat/route.ts` (system prompt inline or `src/lib/prompts/system.ts`) |
| **Output validation** | None — free-form text response |
| **Estimated tokens per call** | Input: ~800 (system prompt + history) / Output: ~200 |
| **Fallback on LLM error** | Surface user-friendly message in Spanish |

**System prompt strategy:** The system prompt is a static string seeded with Paula's real clinic data (hours, services, location, prices). It instructs the model to answer only questions about the clinic and to respond in Argentine Spanish. It does not use dynamic DB injection in v1.
