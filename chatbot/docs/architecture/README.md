# Architecture Decision Records

This directory contains ADRs (Architecture Decision Records) for NeoVet.
Each ADR documents a significant technical decision: what was decided, why, and what alternatives were considered.

## Status values

| Status | Meaning |
|--------|---------|
| `Proposed` | Under consideration, not yet committed |
| `Accepted` | Decision made, not yet implemented |
| `Implemented` | Decision made and fully deployed |
| `Deferred` | Decision made, implementation pending a future phase |
| `Deprecated` | Superseded by a newer ADR |
